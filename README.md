# MiniGame

团结引擎 1.9.3

## 项目结构

```
MiniGame/
├── docs/              # 设计文档、团队分工
│   ├── GAME_DESIGN.md # 游戏设计文档（含剧情、玩法、美术需求）
│   └── TEAM.md        # 团队分工与协作规范
├── 美术资源/           # 角色立绘、场景背景

│   └── 场景/          # 背景：实验室、维修间、走廊
└── README.md
```
