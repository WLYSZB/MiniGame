# MiniGame

小组 MiniGame 大赛参赛项目

## 项目简介

一个2D闯关小游戏，讲述一个垃圾袋误入宇宙特快有限公司，为了弥补过失踏上修复平衡器的旅程。

**游戏类型：** 2D闯关（推箱子/躲避/Boss战混合玩法）

**游戏引擎：** 团结引擎 1.9.3（Unity 2022 LTS）

**游戏风格：** 喜剧吐槽，角色互动驱动剧情

## 如何运行

1. 使用团结引擎 1.9.3 打开 `minigame/` 项目
2. 打开 `Scenes/MainMenu.unity` 或 `Scenes/OpeningScene.unity`
3. 点击 Play 运行
