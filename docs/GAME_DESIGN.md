# 游戏设计文档

## 游戏概述

- **游戏名称**：MiniGame（暂定）
- **类型**：2D闯关（混合玩法：推箱子/解密/射击等）
- **主题**：宇宙流浪生存，遇到星球和NPC，闯关对话
- **引擎**：Unity (C#)

## 角色设定

### 主角
- **形象**：垃圾袋（有自我意识的塑料购物袋）
- **名字**：玩家开局自定义
- **背景**：从天而降，砸坏了行星宇宙平衡器的研究员的装置

### 固定NPC：行星
- **身份**：宇宙平衡器研究员
- **作用**：跟随主角，提供剧情对话
- **性格**：吐槽役

### 新NPC（后续章节）
- 每个星球遇到新NPC
- 可能提供新技能/帮助
- 是否制造麻烦看剧情发展

## 开场流程

1. **CG动画**：垃圾袋从天而降，砸坏装置
2. **起名弹窗**：玩家给主角起名
3. **NPC对话**：与行星的对话
4. **第一关**：推箱子新手教程
5. **完成后**：回到游戏主界面

## 关卡设计

### 第一章：新手教程

- **玩法**：推箱子
- **地图**：3x3 或 4x4
- **箱子数量**：2-3个
- **目标**：箱子推到指定位置（工作台）
- **步数限制**：不限

### 后续章节

- 每个星球 = 一个章节
- 每个星球固定一种玩法
- 每种玩法都有1关新手教程
- 具体设计待后续剧情补充

## 玩法系统

### 推箱子（第一章）
- 玩家控制主角推动箱子
- 箱子推到指定位置即完成
- 不限步数，让玩家熟悉规则

### 解密（后续）
- 待设计

### 射击（后续）
- 根据剧情设定
- 目前不考虑

## 对话系统

- 对话在关卡前后展示
- 以剧情需要为准
- 行星自动触发提示（待后续剧情实现）

## 美术风格

- 主角：可爱的垃圾袋形象
- 场景：宇宙星球风格
- UI：简洁明了

## 验证标准

- 玩家可以完成第一章推箱子关卡
- 没有严重Bug
- 美术资源完整
- 可以正常打包运行
