# 游戏设计文档

## 游戏概述

- **游戏名称**：MiniGame（暂定）
- **类型**：2D闯关（混合玩法：推箱子/躲避/Boss战等）
- **主题**：宇宙流浪生存——一个垃圾袋误入宇宙特快有限公司，为了弥补过失踏上了修复平衡器的旅程
- **引擎**：团结引擎 (Tuanjie / Unity 2022 LTS)
- **风格**：喜剧吐槽，角色互动驱动剧情

## 角色设定

### 主角：垃圾袋
- **形象**：印着"沃Xx"Logo的塑料购物袋，有自我意识
- **名字**：玩家开局自定义
- **背景**：从天而降，穿过裂隙砸在蓝绿色星球上，砸坏了行星的宇宙平衡器
- **性格**：无辜、尴尬、被动卷入

### 固定NPC：行星（宇宙平衡器研究员）
- **身份**：宇宙特快有限公司的宇宙平衡器研究员
- **作用**：跟随主角，提供剧情对话、引导、吐槽
- **性格**：吐槽役、嘴毒但本质善良、对年终奖执念很深
- **口头禅**：关于年终奖的抱怨

### NPC：赫尔墨斯（火球星球）
- **身份**：宇宙特快有限公司员工
- **初登场**：第一章（失控状态，满屋火球和工具）
- **性格**：无语型、捂脸担当
- **特点**：曾用一整套《勇者传说》让卡冈图雅安静下来
- **加入队伍**：第一章完成后

### NPC：卡冈图雅（黑洞）
- **身份**：宇宙特快有限公司最闹腾的成员
- **初登场**：第二章（Boss战）
- **性格**：中二、自称"迷途的勇者"的挑战者、爱玩《勇者传说》角色扮演
- **特点**：黑洞引力，会造成地形变化
- **加入队伍**：第二章完成后

### 世界观：宇宙特快有限公司
- 所有星球都在这里工作，"一辈子"
- 平衡器维持所有星球的理智和秩序
- 平衡器损坏后，所有星球失去理智、发狂
- 备用核心可以修复各星球的异常

## 完整剧情流程

### 序章：垃圾袋，从天而降

**开场CG**：宇宙一片寂静，除了某个垃圾桶被打翻的声音。一个塑料购物袋打着旋穿过裂隙，精准砸在蓝绿色星球上。

**对话**：
1. 主角惊觉自己有了自我意识和身体
2. 发现自己砸坏了行星的宇宙平衡器装置
3. 行星暴怒："哇！我的平衡器！！你个混蛋！"
4. 主角尴尬道歉，行星想到可以让主角帮忙
5. 行星拿出备用核心："把核心放到星球们的工作台上"
6. 主角起名（起名弹窗）

**新手教程关卡**：推箱子
- 玩家控制主角推动蓝色小球（备用核心）到指定位置（工作台）
- 3x3 或 4x4 地图，2-3个箱子
- 不限步数，让玩家熟悉操作

**完成后对话**：
- 主角心想"这不很简单么"
- 行星核善的微笑："相信你已经熟练掌握了，让我们去找其他人吧"

### 第一章：生命之歌（推箱子 + 躲避NPC攻击）

**剧情对话**：
1. 主角问："其他人？"
2. 行星介绍宇宙特快有限公司——所有星球在这里工作一辈子（咬牙切齿）
3. 行星恢复微笑，提议去找赫尔墨斯——"他肯定不希望因为这件事影响了自己的优秀员工光年终奖"
4. 引导到另一个房间

**关卡场景**：
- 房间很热，到处都是发狂的赫尔墨斯的火球和工具
- 行星吐槽："看来某人估计要因为破坏公共设施而失去自己的年终奖了呢"
- 主角吓到："这些火球你可没有说过！"
- 行星："^ ^怎么会呢？"

**战斗关卡**：推箱子 + 躲避火球
- 劲爆音乐响起
- 玩家需要躲避四处飞散的火球
- 同时将备用核心推入指定位置
- 火球有规律的运动轨迹（可预判）
- 地图比序章更大

**完成后对话**：
1. 赫尔墨斯扶着脑袋："呃……发生什么了？"
2. 行星解释情况（主角砸坏了平衡器，所有人失去理智）
3. 赫尔墨斯看到备用核心："该死……我的年终奖……"
4. 行星："来不急想你的年终奖了，再晚一点公司都要炸了"
5. 画面震动效果
6. 赫尔墨斯提议："先去卡冈图雅那吧"

### 第二章：神秘的巨震——黑洞，卡冈图雅（Boss战）

**剧情对话**：
1. 主角："刚刚那是什么？"
2. 赫尔墨斯一脸无语："是卡冈图雅，我们中最闹腾的一个"
3. 又震了一下，三人无语
4. 一行人走到走廊另一端寻找卡冈图雅

**Boss战**：
- 劲爆战斗音乐响起
- 卡冈图雅开场一刀斩下（被躲开）
- 卡冈图雅中二台词："迷途的勇者哦，你也是我的挑战者吗？"
- 主角无语问行星和赫尔墨斯："她一直是这样子的吗？"
- 赫尔墨斯捂脸："我本来是用了一整套《勇者传说》才让她安静下来……"
- 行星把主控推出去："上吧！勇者！去展示你的决心！"

**Boss战机制**：
- **黑洞引力**：特定区域会产生移动，地形会被引力拉扯
- **地形杀**：玩家需要避开被引力移动的地形
- **核心放置**：运用地形变化把备用核心放在指定位置
- **难度较高**：比第一章明显提升

**完成后对话**：
1. 卡冈图雅愣住
2. 赫尔墨斯："醒醒，别发呆了"
3. 卡冈图雅认出众人，打招呼
4. 赫尔墨斯质问："你还记得你答应我了什么吗？"
5. 卡冈图雅："如果不破坏公司公有财产，我将获得一整套《勇者……》"
6. 赫尔墨斯抱胸挑眉
7. 卡冈图雅："抱歉……"（对手指）
8. 行星："往好处想，赫尔墨斯，你省下了一大笔宇宙币"
9. 主角："谁来关心一下我……"
10. 行星："自己关心一下自己吧，你的工作才刚刚开始"

|------|------|------|------|----------|

| 第二章 | 神秘的巨震 | Boss战 | ★★★ | 黑洞引力地形 + 推核心 |

## 玩法系统

### 推箱子（序章、第一章）
- 玩家控制主角推动蓝色小球（备用核心）
- 核心推到指定位置（工作台）即完成目标
- 序章：纯推箱子教学，不限步数
- 第一章：加入火球障碍，需要边躲避边推

### 躲避系统（第一章）
- 火球在场景中有规律运动
- 碰到火球会受伤/失败
- 玩家需要观察火球轨迹，找到安全窗口推动核心
- 火球由失控的赫尔墨斯产生

### Boss战（第二章）
- 卡冈图雅作为Boss
- **黑洞引力**：特定区域的地形会被引力拉扯移动
- **地形杀**：被移动的地形碰到会受伤/失败
- 玩家需要利用地形变化的规律，把核心放到指定位置
- 难度较高，需要多次尝试

## 对话系统

- 对话在关卡前后展示，驱动剧情
- 打字机效果逐字显示
- 支持选项分支（可选）
- 角色立绘 + 名字 + 对话框
- 角色表情变化（行星的核善微笑、赫尔墨斯的捂脸等）

## 美术需求

### 角色
- 主角：可爱的塑料购物袋形象（印有"沃Xx"Logo）
- 行星：星球形象，研究员装扮
- 赫尔墨斯：火球星球形象
- 卡冈图雅：黑洞形象，中二气质

### 场景
- 序章：星球表面，机械装置（平衡器）
- 第一章：赫尔墨斯的房间（炎热、火球、工具散落）
- 第二章：走廊 + 卡冈图雅的领地（黑暗、引力特效）

### UI
- 对话框系统
- 关卡UI（暂停、胜利、失败）
- 起名弹窗
- 主菜单

### 特效
- 火球特效（第一章）
- 黑洞引力特效（第二章）
- 画面震动效果
- CG动画（开场垃圾袋下落）

- 玩家可以完成序章推箱子教程

- 对话系统正常工作，完整剧情流程可跑通
- 没有严重Bug
- 美术资源完整
